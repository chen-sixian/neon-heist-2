# 🌃 Neon Heist: 赛博朋克回合制自动战斗模拟器

这是一个用 Python 编写的回合制自动战斗模拟器。我开发这个项目是为了实践我在计算机科学（CS）课程中学习到的核心**面向对象编程 (OOP)** 概念。

[English Version](README.md) | 中文版

## 🤖 什么是自动战斗模拟器？

这不是一个你需要手动操作的游戏，而是一个你“观看”的游戏。运行程序后，两支自动化的数字小队将根据各自的职业逻辑进行战斗。每场战斗的元素（出手顺序、目标选择等）都是**完全随机**的，所以每次运行你都会看到不同的结果。

运行代码，乱斗就会展开。

## ⚔️ 战斗机制如何运作？
系统预设两支小队（Alpha 队 vs Beta 队）。

每个回合，场上所有存活的单位会被随机打乱出手顺序 —— 确保没有哪一方拥有绝对的先手优势。

每个单位会在敌方小队中随机选择一个目标并释放技能。

如果有单位在回合中途阵亡，系统会自动跳过其行动回合。

当其中一支小队被完全消灭时，战斗结束。

每次运行的结果都不一样，因为底层的 random.shuffle()（打乱顺序）和 random.choice()（随机索敌）让同样的阵容能碰撞出完全不同的战局！
